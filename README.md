# Step School System - Students List Webpage

A simple, static HTML webpage designed to display a list of active students within the **Step School System**. This project serves as a basic template for displaying tabular data with minimal stylized components.

---

## 🚀 Features

* **Dynamic Headers:** Uses HTML `<marquee>` elements for basic scrolling text animations (e.g., welcoming users and scrolling the section title).
* **Structured Data Table:** A clean HTML `<table>` structure containing core student information:
  * **ID:** Unique identification number for each student.
  * **Name:** Full name of the student.
  * **Class:** Current academic grade or level (e.g., Pre9, Step6, Step3, Step2).
  * **Status:** Current enrollment status.
* **External Styling Integration:** Links to an external stylesheet (`style.css`) to separate presentation from content.

---

## 📂 File Structure

```text
├── Index.html      # Main HTML document containing structure and student data
└── style.css       # External stylesheet for custom layout and design

```

## 🛠️ Usage & Deployment

To view the webpage locally:

1. Clone or download this repository.
2. Ensure you have a `style.css` file in the same directory if you wish to apply custom styles.
3. Open `Index.html` in any modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.).

---

## 💡 Future Enhancements

> **Note:** The `<marquee>` tag is considered obsolete in modern web development.

* **Modernize Animations:** Replace the legacy `<marquee>` tags with modern CSS keyframe animations for better web standards compliance and accessibility.
* **Dynamic Data Loading:** Connect to a backend database or an external JSON file to dynamically load student records instead of hardcoding them into the HTML.
* **Interactivity:** Add search and filtering capabilities using JavaScript to quickly find students by Name, Class, or ID.
